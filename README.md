# expense-tracker

Next.js 14+ expense tracking web app (App Router, TypeScript, Tailwind CSS, shadcn/ui, Prisma, JWT auth).

## Stack
- Next.js 14+ (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- PostgreSQL via Prisma ORM
- JWT authentication

## Setup
```bash
npm install
npx prisma generate
npm run dev
```
